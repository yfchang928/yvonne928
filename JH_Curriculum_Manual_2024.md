# 📅 第一場詳細教案（7/6，3 小時）

## 時段 1：破冰 + AI 導入（9:00-9:40，40 分鐘）

### 目標
- 建立心理安全感
- 激發對 AI 的好奇而非恐懼
- 建立課程的核心概念框架

### 活動流程

**破冰活動（5 分鐘）**
```
問題：「你用過 AI 嗎？什麼時候、用來做什麼？」
(不收集答案，只是打破僵局)

預期反應：
- 有些人會說用過 ChatGPT
- 有些人會說不知道怎麼用
- 有些人會說「有點怕」

提示語言：「這堂課，我們把恐懼變成機會。」
```

**視頻引入（3 分鐘）**
- 播放 1-2 分鐘的 AI 教學實例
- 例子：「AI 如何幫老師設計更好的問題」

**核心概念講解（15 分鐘）**
```
講題：「AI 不會替代教師，但會改變教師的角色」

三個重點：
1️⃣ 【層級認知】AI 的五層協作
   - L1 代工（AI 做所有事）→ 不推薦
   - L2 助理（AI 協助，師生做）→ 安全
   - L3 夥伴（AI 與師生協作）→ 高效
   - L4 教練（AI 引導師生學）→ 深化
   - L5 導航（師生用 AI 導航世界）→ 願景

   關鍵問題：「你希望 AI 在課堂中是什麼角色？」

2️⃣ 【危險信號】什麼時候不該用 AI
   - 考試/測驗（不能代做）
   - 需要原創思考的作業
   - 涉及價值判斷的決策

   檢驗方法：「用 AI 後，學生的腦在進步嗎？還是變懶？」

3️⃣ 【教師心態】從「害怕被取代」到「掌握主動權」
   - AI 做不了的：同理心、判斷力、創意、希望
   - 這些才是教師的超能力
   - 用 AI 放大你的人類力量
```

**互動環節（10 分鐘）**
```
小組討論（3 人一組）：
「在你的課堂裡，哪一個教學環節最費時？
 用 AI 能幫上什麼忙？用哪一層協作最合適？」

收集 2-3 組的想法，簡短確認。
(不必深入，只是開啟思考)
```

**小結與過渡（7 分鐘）**
```
總結：「AI 不是威脅，是教學的新工具。
      但怎樣用，決定了效果。
      接下來，我們深入認識 AI 與教學的三種結合方式。」

預告下一段：「Multimodality：為什麼視覺、聲音、文字一起用會更強？」
```

## 時段 2：Multimodality 教學設計（9:40-10:40，60 分鐘）

### 目標
- 理解什麼是 Multimodality（多模態）
- 看見五種模態在課堂中的應用
- 初步思考自己課程的多模態可能性

### 知識講解（15 分鐘）

**什麼是 Multimodality？**
```
定義：「同時使用多種『信息傳遞方式』來教學」

五種模態：
1. 文字 (Text)     - 傳統教材、講義
2. 影像 (Image)    - 圖表、照片、圖解
3. 聲音 (Audio)    - 語音、音樂、對話
4. 動作 (Kinesthetic) - 實驗、模擬、手工
5. 互動 (Interactive)  - 遊戲、投票、討論

問題：「傳統教學用了幾種模態？」
預期答案：主要是文字 + 聲音（老師講）
          少數用圖像
          很少用動作和互動
```

**腦科學依據（8 分鐘）**
```
「為什麼多模態更有效？」

研究證據：
✓ 雙重編碼理論（Paivio）
  文字 + 圖像 → 記憶提升 65%

✓ 認知負荷理論（Sweller）
  模態多樣化 → 大腦不容易過載
  因為不同模態用不同的認知通道

✓ 神經多樣性包容
  多模態設計 → 更多學生能學好
  （視覺型、聽覺型、運動型學習者都有機會）

簡單比喻：
「如果只用文字講數學，有些學生聽不懂。
 加上圖解，他們終於『看懂』了。
 再加上實物操作，他們『摸懂』了。
 這就是多模態的力量。」
```

**AI 與 Multimodality 的結合（7 分鐘）**
```
「AI 怎樣幫助教師設計多模態課程？」

實例 1：AI 生成視覺
- 教師：「給我一張說明分子結構的信息圖」
- AI：1 秒內生成 3 個選項
- 效果：從「純文字說明」→「有圖的說明」

實例 2：AI 設計互動
- 教師：「設計一個互動式的英文文法遊戲」
- AI：提供 Prompt 與工具建議
- 效果：課程從「講述」→「互動式」

實例 3：AI 寫腳本
- 教師：「幫我寫一個 5 分鐘的課堂角色扮演劇本」
- AI：1 分鐘產出
- 效果：從「紙上談兵」→「實際演練」

關鍵態度：
「AI 是設計助手，不是替代品。
 教師決定方向，AI 快速執行。
 這樣教師有更多時間專注於『人』的互動。」
```

### 實踐活動：課程重設計（35 分鐘）

**活動名稱：「把你的課變多模態」**

**第一步：個人反思（8 分鐘）**
```
工作表：「我的課程現狀」
（每位教師填寫）

問題 1：我教什麼科目？
問題 2：目前我的課用了哪些模態？
       ☐ 文字（講義、黑板）
       ☐ 影像（圖片、視頻）
       ☐ 聲音（我的聲音、音樂）
       ☐ 動作（實驗、遊戲、模擬）
       ☐ 互動（討論、投票、小組活動）
問題 3：我最少用的模態是？
問題 4：如果加入那個模態，效果會怎樣？

（給 8 分鐘，不趕）
```

**第二步：小組分享（12 分鐘）**
```
3 人一組，每人 3 分鐘分享

分享順序：
1. 我教什麼
2. 我目前用了哪些模態
3. 我想加的模態

其他人的角色：傾聽 + 提一個建設性問題

例如：
A 說：「我教高中化學，目前主要用文字和圖像」
B 問：「有沒有可能加入實驗操作的影片或 VR 模擬？」
```

**第三步：AI 協作示範（15 分鐘）**
```
講師示範一次完整的「用 AI 重設計課程」流程

情境：將一堂「英文閱讀課」變成多模態

步驟 1：確認目標
「我要教學生閱讀並理解一篇英文短文」

步驟 2：列出現有模態
「現在只用文字教，很多學生昏昏欲睡」

步驟 3：設計多模態組合
「文字 + 影像（短文背景圖）+ 聲音（朗讀）+ 互動（小組討論）+ 動作（改編成劇本表演）」

步驟 4：用 AI 產出資源
提示詞範例：
───────────────────────────────────
Prompt: 「我要設計一堂英文閱讀課，主題是『環境保護』。
能否為我：
1. 生成 3 張信息圖，視覺化環保概念
2. 寫一個 2 分鐘的英文朗讀稿
3. 設計一個 15 分鐘的小組討論題組
4. 提供 2 個課堂活動點子（用到身體動作）
」
───────────────────────────────────

步驟 5：整合與使用
「現在我有了多模態資源，組織成 45 分鐘的課程」

步驟 6：評估與迭代
「哪個部分有效？哪個需要調整？下堂課改進」

講師示範要點：
- 講得很慢（因為聽力不佳）
- 用圖片表達，不只文字
- 展示實際的 AI 產出（截圖或列印）
- 強調這是「人 + AI」協作，不是 AI 替代
```

**小結（3 分鐘）**
```
「Multimodality + AI，改變了什麼？

傳統：
教師準備 + 教室講述 + 學生記筆記
        ↓
        很多模態被遺漏了

新方式：
教師設定方向 + AI 協助產資源 + 多模態呈現 + 學生深度參與
        ↓
        每個學生都能用自己最舒適的方式學

重點：教師節省了製作時間，有更多精力在『人』的互動。」
```

## 時段 3：SEL × EPOCH 認識 + 教師自評（10:50-11:50，60 分鐘）

### 目標
- 理解 SEL（社情學習）的五個維度（EPOCH）
- 教師自我評估，找到發展方向
- 看見如何在課堂中融入 SEL

### 知識講解（15 分鐘）

**什麼是 SEL (Social-Emotional Learning)？**
```
定義：「幫助學習者發展五個核心的人類能力」

在傳統教育中，我們專注於：
- 數學、語言、科學（技能型）

在 SEL 中，我們發展：
- 同理心、存在感、判斷力、創造力、希望（人類型）

為什麼重要？
✓ 研究證實：SEL 強的學生，學業表現也更好
✓ 工作市場需求：AI 能做技能，但做不了『人類能力』
✓ 人生幸福度：有 EPOCH 的人，人生滿足感更高
```

**EPOCH 五力詳解（13 分鐘）**

```
E - Empathy（同理心）
    定義：能真實感受他人的感受，並做出回應
    
    教室應用例子：
    ✗ 不好：學生互相指責，不聽彼此
    ✓ 好：學生能說出同學的想法，尊重不同意見
    
    如何培養？
    - 講故事後問「主人翁什麼感受？」
    - 小組討論強制傾聽規則
    - 同理心寫作練習（換位思考）
    
    AI 的角色：
    - AI 無法真實同理（這是人類獨有的）
    - 但 AI 可以示範同理語言，讓學生學習表達

─────────────────────────────────

P - Presence（存在感）
    定義：在社群中感受到被看見、被接納、有歸屬感
    
    教室應用例子：
    ✗ 不好：學生上課時很孤單，不覺得自己重要
    ✓ 好：學生知道自己的意見被重視，有存在感
    
    如何培養？
    - 全班活動中讓每個學生有發言機會
    - 教師記得學生的名字和想法
    - 建立「安全說話」的教室文化
    - 小組活動讓學生互相認識
    
    AI 的角色：
    - AI 無法提供真實的『被看見』
    - 但可以處理行政工作，讓教師有更多時間陪伴學生

─────────────────────────────────

O - Opinion（判斷力/觀點）
    定義：能獨立思考，有自己的觀點，同時尊重他人
    
    教室應用例子：
    ✗ 不好：學生只會接受老師的答案，不敢有不同想法
    ✓ 好：學生會說「我的想法是...」並願意聽別人的
    
    如何培養？
    - 鼓勵學生問「為什麼」
    - 討論時提出多元觀點
    - 在道德議題上有深度對話
    - 尊重學生的異議（即使錯誤）
    
    AI 的角色：
    - AI 可以提供多個觀點，幫助學生思考
    - AI 可以當「辯論對手」，訓練判斷力
    - 但最後決定權在學生

─────────────────────────────────

C - Creativity（創造力）
    定義：能產出新想法、新作品，表達自己的獨特性
    
    教室應用例子：
    ✗ 不好：所有學生做一樣的作業，沒有表達空間
    ✓ 好：學生能用自己的方式呈現學習，有創意空間
    
    如何培養？
    - 開放式任務（不只有一個正確答案）
    - 鼓勵試驗和失敗
    - 展示學生作品
    - 跨領域連結（這堂課的概念能用在哪個領域？）
    
    AI 的角色：
    - AI 可以示範創意（生成故事、視覺、樂曲）
    - AI 可以當『創意夥伴』（「這個想法怎樣改進？」）
    - 但學生的創意不能被 AI 代做

─────────────────────────────────

H - Hope（希望力/願景力）
    定義：對未來有正面的期待，相信自己能成長
    
    教室應用例子：
    ✗ 不好：學生對未來迷茫，覺得「無論如何都做不好」
    ✓ 好：學生有清晰的短期目標，相信自己能進步
    
    如何培養？
    - 在課堂中展示「進步」的例子
    - 幫學生設定具體而可達的目標
    - 談論對未來的願景（不是恐懼）
    - 當他們遇到困難時，幫他們看見可能性
    
    AI 的角色：
    - AI 可以協助設計成長計畫
    - AI 可以鼓勵性的語言回應
    - 但真實的『希望』來自人與人的連結
```

**為什麼教師自己要先有 EPOCH？（2 分鐘）**
```
「教師是課堂文化的示範者。
 如果教師有同理心，課堂就溫暖。
 如果教師有創意，課堂就生動。
 如果教師有希望，學生就相信未來。

 你有什麼，才能給學生什麼。」
```

### 實踐活動：EPOCH 自評 + 分享（35 分鐘）

**第一步：個人評估（8 分鐘）**
```
工作表：「我的 EPOCH 五力自評」

指示：「用 1-5 分評估自己」
      1 = 很弱，2 = 有點弱，3 = 中等，4 = 很強，5 = 特別強

E - Empathy（同理心）
    我能真實感受學生的感受嗎？
    □ 1  □ 2  □ 3  □ 4  □ 5

P - Presence（存在感）
    學生在我的課堂中感到被看見嗎？
    □ 1  □ 2  □ 3  □ 4  □ 5

O - Opinion（判斷力）
    我有自己的教學哲學嗎？會鼓勵學生獨立思考嗎？
    □ 1  □ 2  □ 3  □ 4  □ 5

C - Creativity（創造力）
    我的課程有創意嗎？會給學生創新的空間嗎？
    □ 1  □ 2  □ 3  □ 4  □ 5

H - Hope（希望力）
    我對教學的未來樂觀嗎？會激勵學生？
    □ 1  □ 2  □ 3  □ 4  □ 5

總分：___ / 25

附加問題：
「你分數最高的是哪一力？（你的超能力）」
「你分數最低的是哪一力？（你的成長機會）」
```

**第二步：小組討論（15 分鐘）**
```
分組：4-5 人一組

輪流分享（每人 2 分鐘）：
1. 我的超能力是 ___，因為...
2. 我想加強的是 ___，因為...
3. 我可以怎樣改進？

其他人的角色：
- 傾聽
- 提一個具體的支持建議
- （例：「你可以試試看在課堂裡多問『你怎麼想？』」）

講師在場走動，聽取 2-3 組的分享，簡短回應。
```

**第三步：視覺化展示（10 分鐘）**
```
講師展示 EPOCH 五力的「成長路徑」

例子：「一個教師從『同理心 2→4』的故事」
1. 起點：不太能傾聽學生，常打斷
2. 意識：發現學生在課堂中很沉默
3. 嘗試：強制自己傾聽 5 分鐘（不插話）
4. 發現：學生開始說話了！
5. 進化：傾聽變成習慣，同理心提升到 4
6. 影響：整個課堂氣氛改變了

重點：「改變需要時間和意識。
      但每一小步都會帶來課堂變化。」

視覺輔助：
用圖表展示一個五邊形（EPOCH 能力雷達）
- 每個維度一條邊
- 教師的分數決定了形狀
- 越均衡的五邊形，課堂效果越好
```

**小結（2 分鐘）**
```
「你剛才做的，就是教師的 SEL 實踐。
 你評估了自己，找到了優勢和成長空間。
 
 接下來的課程，我們會用具體工具來追蹤和提升這五力。
 最重要的是：定期反思，小步進展。」
```

## 時段 4：課程小作品 + 反思（11:50-12:00，10 分鐘）

### 目標
- 整合三小時的學習
- 產出可帶回學校的小作品
- 建立持續改進的意識

### 活動：「我的課程改進方案」（10 分鐘）

**表格形式：**
```
【我的課程改進方案 - 帶回學校版本】

我的課程名稱：_______________

目前使用的模態：
☐ 文字  ☐ 影像  ☐ 聲音  ☐ 動作  ☐ 互動

我最想加入的模態：_______________

我的 EPOCH 超能力：_______________

我想改善的 EPOCH 面向：_______________

具體行動（下學期試試看）：
1. _______________
2. _______________
3. _______________

我需要的幫助：
☐ AI 工具的使用方法

# 📅 第二場詳細教案（7/8，2 小時）

## 課程背景
- 學生已完成第一場（AI、Multimodality、SEL 基礎知識）
- 現在要從「認識」進展到「設計」和「試教」
- 輸出：一個完整的「AI-SEL 多模態課程」原型 + 試教

## 時段 1：回顧 + 分組（1:00-1:30，30 分鐘）

### 目標
- 快速連接第一場的學習
- 形成異質性小組（每組 4-5 人，涵蓋不同科目）
- 建立共備的工作框架

### 活動流程

**快速回顧（5 分鐘）**
```
講師用 3 張圖片/投影片快速回顧：

1️⃣ AI 的五層協作
   (只強調第二層「助理」對教師最安全)

2️⃣ Multimodality 五種模態
   (文、圖、音、動作、互動)

3️⃣ EPOCH 五力
   (同理、存在、判斷、創造、希望)

提問：「誰還記得？」（舉手 2-3 人，快速確認）
```

**分組原則（5 分鐘）**
```
目標：每組有不同科目的教師，增進跨領域對話

方法：
講師說：「站起來，我們用『科目分組』方式。」

科目分類：
- 組 1：英文、語文類（約 4-5 人）
- 組 2：數學、自然科學類（約 4-5 人）
- 組 3：社會、藝術類（約 4-5 人）
- 組 4：體育、生活課程類（約 4-5 人）
  （根據實際人數和科目調整）

確認分組後坐到指定的「工作區」
（每組一張大桌子，中間放工作物料）
```

**共備框架說明（10 分鐘）**
```
講師展示「共備工作表」（投影在螢幕上，也發紙本）

【小組共備工作表】

步驟 1：確認課程主題（10 分鐘）
  「我們要設計什麼課程？」
  - 可選自己的學科內容
  - 可跨領域設計
  - 主題需要是 40-45 分鐘的課堂

步驟 2：融入 AI（10 分鐘）
  「在哪個環節用 AI？用什麼層級？」
  - L2 助理（推薦）
  - L3 夥伴
  
步驟 3：設計 Multimodality（10 分鐘）
  「用五種模態中的哪些？怎樣呈現？」
  文字：講義、黑板
  影像：圖表、圖解、視頻
  聲音：學生朗讀、對話
  動作：實驗、角色扮演、手工
  互動：討論、投票、遊戲

步驟 4：融入 EPOCH 一個維度（10 分鐘）
  「這堂課最能培養哪一個 EPOCH 力？」
  選一個（不要全部）
  
步驟 5：寫出試教計畫（10 分鐘）
  「這堂課怎樣教？教多長？」
  設計 8-10 分鐘的試教部分

時間表：
準備時間：30 分鐘（含這個說明 10 分鐘）
試教時間：60 分鐘（每組 12-15 分鐘）
反思：30 分鐘
```

**物料準備（10 分鐘）**
```
給每組準備：
- A3 工作紙（已印刷工作表）
- 便簽紙和色筆
- 電腦/iPad 一台（可選，查資料或用 AI）
- 教材庫（備選教學素材，如圖片、短文、影片清單）

講師分發時說：「這些工具幫助你們快速設計。
                別被完美嚇到，20-30 分鐘內完成『草稿』就好。
                試教會暴露問題，我們現場就調整。」
```

## 時段 2：小組共備 + 試教準備（1:30-2:30，60 分鐘）

**分成兩個循環：**

### 循環 A：前 30 分鐘小組工作坊

**小組工作流程（30 分鐘）**

**第 1 步：決定課程主題（5 分鐘）**
```
工作表提示：「選擇一個 40-45 分鐘課程的主題」

範例引導：
- 英文課：「環保議題閱讀」
- 數學課：「分數的應用」
- 社會課：「民主制度」
- 自然課：「光的折射」

組長（由組員推薦）紀錄決定。
```

**第 2 步：用 AI 快速生成資源（8 分鐘）**
```
組員（一人負責）在電腦上：
打開 ChatGPT 或其他 AI
輸入簡單 Prompt：

「我要設計一堂課，主題是『[主題]』。
  對象是國中生。
  40 分鐘。
  請給我：
  1. 這個主題的 3 個核心概念
  2. 一個吸引學生的開場問題
  3. 一個實踐活動的點子
  4. 一個評量方式」

AI 回應 30 秒內產出。
組員列印或截圖帶回桌子。

其他人同時進行步驟 3。
```

**第 3 步：設計 Multimodal 呈現（10 分鐘）**
```
全組討論，用便簽紙畫出課程流程：

時間 | 學習活動 | 用什麼模態 | AI 協助？
───────────────────────────────────
0-5min | 開場問題 | 聲音+文字 | (無)
5-15min | 講解核心概念 | 文字+影像 | AI 生成圖
15-25min | 小組討論 | 互動+聲音 | (無)
25-40min | 創意表達 | 多模態 | AI 協助檢查
40-45min | 反思和回家功課 | 文字 | (無)

講師提醒：「不要複雜。簡單試教就可以。」
```

**第 4 步：加入一個 EPOCH 維度（5 分鐘）**
```
討論：「這堂課最能培養哪一個 EPOCH 力？」

例子映射：
課程內容 | 最可能培養的 EPOCH
────────────────────
環保議題 | 同理心（care for planet）
民主制度 | 判斷力（多元觀點）
分數應用 | 希望力（掌握生活技能）
光的折射 | 創造力（發現現象的美）

組員在工作表上勾選並說明。
```

**第 5 步：寫試教計畫（2 分鐘）**
```
決定：「我們試教哪個部分？（8-10 分鐘）」

通常選「中間最精彩的環節」。

在工作表上簡單寫下：
試教環節：_______________
試教時間：8-10 分鐘
試教流程：
1. ...
2. ...
3. ...
```

### 循環 B：後 30 分鐘試教 + 講師巡視

**每組輪流試教（30 分鐘共 4-5 組，每組 6-8 分鐘）**

```
安排：
- 時間：1:30-2:00pm（第一輪試教）
  第 1 組：1:30-1:38
  第 2 組：1:39-1:47
  第 3 組：1:48-1:56
  第 4 組：1:57-2:05

試教程序：
1. 一個組員（代表）上台示教 6-8 分鐘
   其他 20 個教師和講師是「學生」
   
2. 代表用簡單道具/投影/口頭展示課程

3. 試教後（1 分鐘）：
   同組的人舉 1-2 個「學生問題」
   代表示範怎樣回應
   （這展示了互動性）

4. 其他組的教師觀察，記下想法

講師在旁邊觀察，筆記要點，等等用來反思。
```

**講師在試教時的角色：**
```
- 不打斷（讓他們完成展示）
- 仔細看哪裡有效、哪裡卡住
- 記下 AI 使用、多模態呈現、EPOCH 融入的情況
- 準備一句鼓勵 + 一個建設性問題
```

## 時段 3：反思與檢視（2:30-3:00pm，30 分鐘）

### 目標
- 從試教中學習
- 看見「設計」和「實踐」的差距
- 建立帶回學校的具體計畫

### 活動流程

**第一部分：集體反思（15 分鐘）**

**講師引導全體討論：**
```
問題 1：「試教時，哪個環節最順暢？」
（舉手 3-4 人，講師指導發言）

回答範例：
「我們的小組討論環節，學生（觀眾）都很投入」

講師追問：「為什麼有效？」
回答可能：「因為有具體問題，時間限制明確」

講師：「對。這就是多模態的力量。
      有互動 + 聲音 + 時間框架，就能抓住注意力。」
───────────────────────────────────

問題 2：「哪個環節卡住了或不夠清楚？」
（舉手 2-3 人）

回答範例：
「我們用 AI 生成的圖有點複雜，
 學生（觀眾）需要更多時間理解」

講師：「很好的觀察。這個問題怎樣解決？」
回答可能：「加上語音說明，或分成幾張簡單圖」

講師確認：「完全同意。這就是『迭代』——
           試過一次，馬上改進。」
───────────────────────────────────

問題 3：「有沒有看到 EPOCH 力量的展現？」
（舉手 2-3 人，如果回答難，講師幫提示）

例子提示：
「第 3 組的課，學生需要兼聽彼此的想法，
 這培養了什麼 EPOCH 力？」
回答：「同理心」

講師：「對！好設計自然帶出人類能力。
      不需要『額外加』，而是課程本身就有。」
```

**第二部分：小組個別回饋（10 分鐘）**

```
講師（或助教）對每組 1-2 分鐘反饋

反饋框架：
1. 一句鼓勵（你們的 __ 部分很棒）
2. 一個具體觀察（我看到學生在 __ 時投入）
3. 一個改進建議（如果下次試試 __ ，會更好）
4. 一個鼓勵行動（你們可以在學校試試看這個）

例子：
「第 2 組，我很欣賞你們的『分數應用』主題選擇。
 當你問『這個比例在日常生活哪裡用得到』時，
 全班學生都舉手了——這是高效的互動！
 
 如果下次試試看『學生找出自己生活中的例子』，
 會更有創意空間。
 
 回學校後，用這個框架設計一堂課，
 看看效果怎樣。我很期待聽到你們的故事。」
```

**第三部分：帶回學校行動計畫（5 分鐘）**

```
每位教師最後填寫：

【我的課程改進計畫 - 帶回學校】

從第一場和第二場的共備，我學到：
☐ 多模態設計可以這樣用：_______________
☐ AI 可以幫我：_______________
☐ 我的 EPOCH 超能力是：_______________

我決定試試看的改變：
1. 下周試一個課程改進（具體是什麼）：_______________
2. 預期效果（學生會怎樣變化）：_______________
3. 我需要的支持：_______________

試教日期：_______________
試教結果反饋給講師：_______________
```

**講師最後小結（2 分鐘）**
```
「今天你們做的，是真實的教育創新。
 
 從『認識 AI』到『用 AI 設計課程』到『試教看效果』
 這就是『實踐→反思→改進』的循環。
 
 很多教師講完課就完了。

 AI 會變，技術會變，但『人』是教育的中心。
 用 AI 放大你的人類力量（同理心、創意、判斷力）

# 💬 講師備忘與提示語

## 語言調整要點
```
因為教師的聽力和口說不佳：

1. 說得慢（比正常速度慢 20-30%）
2. 用簡單單字（避免複雜的教育術語的英文表達）
3. 常用視覺輔助
4. 提供講義（他們可以讀而不只聽）
5. 重要概念用中文 + 簡單英文雙語呈現
6. 問題清楚明確，避免開放式太廣的問題（容易不知道怎麼回答）
```

## 互動時的鼓勵語
```
鼓勵第一場的互動：
「沒有完美答案。分享你的想法就好。」
「如果不同意，那很好——我想聽你的想法。」

鼓勵第二場的試教：
「試教會暴露問題。暴露問題是好事，代表我們看到改進空間。」
「不用演得像『真的上課』。就像『想像教室』一樣示範就行。」
```

## 常見的教師疑慮與回應
```
Q: 「我不懂 AI，怎樣教學生用？」
A: 「今天我們不要求你變成 AI 專家。
    只要知道：AI 是個幫手，能快速生成想法。
    你設定方向，AI 執行。
    就像用搜尋引擎一樣簡單。
    我會示範整個過程。」

Q: 「怎樣確保學生不會『讓 AI 做功課』？」
A: 「很好的擔心。秘訣是：
    設定清楚的『AI 紅綠燈』。
    紅燈：學生自己做（考試、需要原創思考）
    綠燈：AI 協助（查資料、給回饋、示範方法）
    教師的工作是幫學生分辨差別。
    我等等會給你一個工具。」

Q: 「這需要花多少時間準備？」
A: 「如果用 AI 協助，比從零開始快 10 倍。
    3 小時課程，你可能只需要 30 分鐘設計。
    用 AI 生成框架，你調整細節。
    大大省時。」
```
